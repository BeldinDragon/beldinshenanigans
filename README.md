# Current Features

### Bismuth Collection
- Bismuth item, ores and block.
- **Chisel**: Swap stone blocks with their variants using right click.
- **Magic Block**: A bismuth block transforming bismuth into diamond. Don't step on it!

### Ingredients
- Obsidian Sticks
